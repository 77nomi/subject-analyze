# 项目介绍

  一个基于**uniapp+vue3**专业学习规划分析平台，本代码主要为移动端。  

- dev-uniapp为开发分支

- dev-master为最新的打包分支，仅供记录使用

- package为as的打包分支，仅供打包使用

- dev-kotlin为kotlin分支，已停止更新

- feat-picker为选择器更换分支，已进行更换，并入主分支
